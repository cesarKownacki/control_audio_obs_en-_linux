# 🧪 AudioSuite - Laboratorio de Desarrollo (Legacy)

Este repositorio contiene el historial evolutivo de **AudioSuite**. Aquí se puede observar el proceso de aprendizaje, desde la automatización básica hasta la optimización de software de escritorio.

## 📈 Línea de Tiempo Técnica:

* **V1 (Bash):** Automatización pura mediante scripts de shell para carga de módulos `null-sink`.
* **V2 (Python CLI):** Transición a lógica de objetos y manejo de procesos con `subprocess`.
* **V3 (GUI Alpha):** Implementación de interfaz gráfica con `CustomTkinter`. 
* **V4 (Performance Edition):** Optimización final con sistema de Caching para hardware legacy.
